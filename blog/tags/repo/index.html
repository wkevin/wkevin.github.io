几个 github 项目选型的比较

March 18, 2024
wKevin
一颗向上的水滴

最近有个新项目，要重新选型，但发现 https://www.githubcompare.com/ 频频异常，就自己写了个脚本，从 https://api.github.com/repos/... 下获取数据，然后抓取出自己关心的字段，再做横向对比。

数据抓取结果如下：

低代码引擎​

ProjectName     CreateDate  LatestPush  Star/k  Fork/k  Size/M  License     Repo
jeecg-boot      2018-11(5)  2024-03-18  37.3    14.2    58.4    apache-2.0  jeecgboot/jeecg-boot
lowcode-engine  2021-12(2)  2024-03-13  13.7    2.4     5.9     mit         alibaba/lowcode-engine
illa-builder    2022-04(2)  2024-03-08  10.5    1.0     53.7    apache-2.0  illacloud/illa-builder
tmagic-editor   2022-02(2)  2024-03-15  4.0     0.7     29.8    other       Tencent/tmagic-editor
-------- 采样时间: 2024-03-18
  • jeecgboot/jeecg-boot 前后端分离架构 SpringBoot2.x，SpringCloud，Ant Design&Vue，Mybatis-plus，Shiro，JWT，支持微服务。包含：Online 表单开发、Online 报表、报表配置能力、在线图表设计、大屏设计、移动配置能力、表单设计器、在线设计流程、流程自动化配置、插件能力（可插拔）等组件。
  • alibaba/lowcode-engine 源自阿里，材料系统、设置器、插件等开箱即用，还有一个完整的工具链。
  • illacloud/illa-builder 包含：仪表板，crud 应用程序，管理面板，crm, cms 等。支持 PostgreSQL, MySQL, Supabase, GraphQL, MongoDB, MSSQL, Rest API, hug Face, Redis 等。自动化工作流程与时间表或 webhook
  • Tencent/tmagic-editor 源自腾讯，主要定位是搭建生成移动端 H5 页面
